# InvoiceKit — Full Website Prompt

## Concept & Name
InvoiceKit — a free, no-login invoicing tool. The entire promise lives in one sentence: "Pick a template. Fill it in. Download your invoice." No accounts. No friction. No clutter.

---

## User Flow

Land on Hero → Scroll to Templates → Click "Use Template"
→ Side Panel opens → User fills in fields → Live Preview updates
→ Click "Download PDF" → File saves instantly → Done.

Zero pages. Zero navigation. Zero login. Everything happens on one long scrollable page with a slide-in editor panel.

## Key UI Rules Throughout
- No modals. No popups. No cookie banners. No newsletter signup interruptions.
- No skeleton loaders — everything is client-side, instant.
- Every interactive element has a hover state, an active/pressed state (slight scale-down), and a focus ring in Slate Blue for accessibility.
- Spacing system: 4px base grid. All padding/margin in multiples of 4.
- Responsive: Three breakpoints — mobile (<640px), tablet (640–1024px), desktop (>1024px). Template grid: 1 col → 2 col → 2 col. Editor panel: full screen → full screen → side drawer.

## Why This Color Scheme Works for Invoicing
Slate Blue reads as trustworthy and financial (think banking apps, DocuSign, QuickBooks). The warm white background (#F8F7F4) is a deliberate paper metaphor — users subconsciously feel like they're already looking at a document. Teal is reserved exclusively for download/success actions, so it always means "you're getting something."

## The Biggest Trust Signal
The phrase "Your data never leaves your browser" under the download button. For a financial tool with no login, users will wonder where their client names and amounts go. Answering that proactively — at the exact moment they're about to click download — removes the last hesitation.
